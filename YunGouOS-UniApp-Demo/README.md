# uniapp使用YunGouOS示例代码

#### 介绍

通过uniapp使用YunGouOS支付SDK，一行代码完成支付接入，支持个人、个体户、企业全渠道接入官方支付接口

集成了微信扫码支付、小程序支付、微信APP支付、支付宝APP支付、微信H5支付接口

所有接口均支持个人用户接入，需开通个人微信商户号和支付宝商户号
